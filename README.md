# log.nikhil.io

[My personal log](https://log.nikhil.io/) powered by [11ty.js](https://www.11ty.dev/).

```bash
# Install dependencies
yarn

# Make a new post
yarn new

# Start a live-reloading server (on port 9000)
yarn start

# Clean build
yarn clean

# Build site
yarn build

# Publish stuff
yarn sync
```
