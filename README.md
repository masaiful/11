# log.nikhil.io

My personal log powered by [11ty.js](https://www.11ty.dev/)

### Usage

Customization in `_data/site.json` and `_scripts/config.js`.

```bash
# Install dependencies
yarn

# Make a new post
yarn new

# Start a live-reloading server (on port 9000)
yarn start

# Clean build
yarn clean

# Build site
yarn build
```
